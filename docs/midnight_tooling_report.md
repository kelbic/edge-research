## M-T2: break-even-модель рампа [ядро; exit-квоты РЕАЛЬНЫМ роутером]

Формула (Midnight.sol:687, встроен урок SVR Gate 1 — exit квотой, не оракулом):
```
seized(t)   = D·lif(t)/WAD·1e36/price
proceeds(t) = uniV3_quote(collateral→loan, seized(t))   # реальный Base-роутер
π(t)        = proceeds(t) − D − gas − flash
t*          = min{t : π(t) > 0}
```
Exit-роутер — Uniswap V3 QuoterV2 на Base (`0x3d4e44Eb…B76a`, проверен он-чейн:
перебор маршрутов coll→loan / coll→WETH→loan / coll→USDC→loan / coll→WETH→USDC→loan
× fee-тиры, по лучшему выходу; маршрут разрешается 1 раз на клип и переиспользуется
по рампу — seized меняется ≤~4%).

**Ключевой результат — exit-импакт при maxLif vs размер клипа** [P: живые квоты
QuoterV2 2026-07-12; бонус maxLif на lltv 0.86 = 4.38%, окно выигрышно если импакт < бонуса]:

| рынок (коллатерал) | $10k | $100k | $1M |
|---|---|---|---|
| **WETH/USDC** | 0.06% ✓ | 0.19% ✓ | 0.26% ✓ |
| **cbBTC/USDC** | 0.04% ✓ | 0.21% ✓ | 1.02% ✓ |
| MORPHO/USDC | 1.30% ✓ | 8.51% ✗ | 87.5% ✗ |
| WELL/USDC | 92.5% ✗* | 99.3% ✗* | 99.9% ✗* |

Пример полной кривой (cbBTC/USDC, D=$50k, газ $0.011): **t\* = +120 c (2.0 мин)**
после maturity, π(t\*) = +$22.46, seized 0.7847 cbBTC.

**Вывод M-T2 (расслоение по качеству выхода — прямой аналог решётки #1 SVR)**:
- **blue-chip коллатерал (WETH, cbBTC)**: exit-ликвидность Base глубокая; окно
  экономически выигрышно на любом реалистичном клипе (импакт ≪ 4.38% бонуса) — t\*
  в первые минуты рампа. Связывающего exit-барьера НЕТ.
- **mid-cap (MORPHO)**: выигрышно до ~$10–30k, умирает к $100k (импакт > бонуса).
- **thin-cap (WELL)**: по Uni-only убыточно уже на $10k.

*⚠️ Оговорка честности: роутер — Uniswap V3 ONLY. На Base доминирует Aerodrome;
WELL и частично MORPHO ликвиднее там. Поэтому импакт mid/thin-cap здесь — **верхняя
граница** (истинный best-exit через Aerodrome лучше). Расширение роутера на Aerodrome —
пункт бэклога (не меняет вывод по WETH/cbBTC: они и на Uni глубоки). Это тот же
методологический принцип, что в SVR Gate 1 (реальный роутер, не оракул).*

Экономический смысл для тайминга: на blue-chip рынках связывающим будет НЕ exit
(его нет), а **FCFS-гонка на границе t\*** (тай-брейк латентностью) + размер приза
(borrow). Оба — не решены капиталом; см. §5-пороги.
