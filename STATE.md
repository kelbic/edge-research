# edge-research — поиск незанятых щелей

## Метод (архетип edge)
Дешёвая заблаговременная позиция под редкое событие, которое рынок лениво/неточно
оценивает. Фильтр кандидата: (1) профи ещё не пришли, (2) нет структурного барьера,
убивающего экономику (как ликвидность выхода убила Monad).

## Дисциплина
- Gate 0 первым: проверить структурный барьер ДО идей исполнения.
- Read-only разведка, ноль капитала, пока edge не доказан на данных.
- Честный NO дешевле ложного GO. Портфель отрицательных результатов с редким золотом.
