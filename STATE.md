# edge-research — поиск незанятых щелей

## Метод (архетип edge)
Дешёвая заблаговременная позиция под редкое событие, которое рынок лениво/неточно
оценивает. Фильтр кандидата: (1) профи ещё не пришли, (2) нет структурного барьера,
убивающего экономику (как ликвидность выхода убила Monad).

## Дисциплина
- Gate 0 первым: проверить структурный барьер ДО идей исполнения.
- Read-only разведка, ноль капитала, пока edge не доказан на данных.
- Честный NO дешевле ложного GO. Портфель отрицательных результатов с редким золотом.

### Почему Nexus — GO (ядро edge, всё верифицировано 3-0)

1. **Сектор = монополия Nexus.** DeFi-insurance категория на DeFiLlama: Nexus $82.5M TVL
   (~84% сектора), runner-up Ease $3.8M (в 22 раза меньше). Перепроверено прямым запросом
   `api.llama.fi/protocols` — цифры сошлись до доллара.
2. **Ленивое ценообразование — механический факт, не гипотеза.**
   `spotPrice = MAX(bumpedPrice − priceDrop, targetPrice)`; никакого внешнего риск-фида;
   цена растёт ТОЛЬКО когда спрос выедает ёмкость; распад повышенной цены обратно к floor —
   **2%/день** (`PRICE_CHANGE_PER_DAY = 2_00` в StakingProducts.sol, сверено доки+код).
   В master-коде bump всего 0.05% за 1% ёмкости — спрос двигает цену ещё слабее, чем по докам.
   Если риск растёт раньше спроса — кавер остаётся дешёвым. Это точный архетип FTX-2022
   (Nexus по FTX реально выплатил $4.92M, 21/24 клеймов одобрено).
3. **Ёмкость покупаема (прямой замер API 2026-07-05):** 234 продукта с ненулевой ёмкостью,
   суммарно ~$1.3B; ставки 0.25–5.4% годовых. Релевантные архетипу листинги:
   Sky USDS Depeg — 0.5%/год, ёмкость $13.4M; USDe/USDG (Kamino) — 4.6%/год, $11.8M;
   CEX-custody: HTX — 3.96%/год, $18.1M; BingX — 4.27%/год, $18.1M. Позиция $50–100k
   от ёмкости листинга — единицы процентов, цену не сдвинет.
4. **Экономика продавца жирная = покупатель в среднем переплачивает:** за 2025 Nexus
   собрал $5.7M премий, выплатил ~$370K (loss ratio ~15:1). Edge может быть только
   ТОЧЕЧНЫМ (выбор момента/листинга), не «покупай всё дешёвое».

### Барьеры (не убивают, но входят в цену edge)

- **KYC-членство**, 27 закрытых юрисдикций (вкл. РФ, Китай, Индию, Японию). Смягчение:
  дистрибьюторы (OpenCover) фронтят покупку без прямого членства — работоспособность
  канала на практике НЕ верифицирована.
- **Выплаты дискреционные, НЕ параметрика.** Почти все типы продуктов `claimMethod=0`
  (индивидуальный клейм с голосованием); в v3 — комитет 3 экспертов, 2-из-3, 72ч + апелляция.
  Платят реально (FTX $4.92M, UST, Arcadia ~$250K, Stream Finance ~$95K), но на спорном
  tail-событии несём assessor-риск.
- **Custody Cover: >90 дней невывода** до права на клейм (FTX: halt 2022-11-08 →
  подача 2023-02-06). Реализация edge ≈ квартал лока капитала + off-chain proof-of-loss.
- **Поведенческое допущение (ГЛАВНЫЙ РИСК ТЕЗИСА):** пул-менеджер может в любой момент
  вручную поднять targetPrice или выдернуть stake; команда Nexus гоняет актуарную модель
  (фев-2025: ею репрайсили 11 протоколов — правда, ВНИЗ). Edge жив, только пока менеджеры
  медленные. Это не структурная гарантия — это то, что меряет Gate 1.

### Урок Y2K (мёртвый «идеальный» дизайн)

Y2K имел максимально ленивую цену (pari-mutuel: P_k = φ_k/(1−c), ноль риск-модели) И
полную параметрику (авто-триггер по Chainlink/Pyth). Но: (а) депозит только в первые
2 дня эпохи, защита в окне депозита НЕ активна — «купить прямо перед событием»
заблокировано дизайном; (б) протокол умер (TVL $31.9k). Вывод: чистая комбинация
«параметрика + ленивая цена» не прокормила двустороннюю ликвидность. Покупательский
edge в неё встроен слишком явно — LP разбежались.

### Что НЕ отвечено (дыры Gate 0)

- **Конкуренция профи** (вопрос 6): ни одно утверждение о фондах, систематически
  скупающих кавер перед событиями, верификацию не пережило. Неизвестно ≠ нет.
- **Вторичный рынок / выход из позиции** Nexus-кавера: refuted/unverified. Считаем
  позицию hold-to-expiry, пока не доказано иное.
- Точный bump-параметр (доки 0.2% vs код 0.05% за 1% ёмкости) — расхождение в пользу
  «ленивости», но требует фиксации по деплойнутому байткоду.
- Работоспособность OpenCover-канала без KYC на практике.

### Gate 1 — что мерить дальше (по убыванию решающести)

1. **Историческая латентность репрайсинга** — главное измерение, решает жив ли edge:
   по он-чейн событиям StakingProducts/StakingPool восстановить, как быстро менеджеры
   поднимали targetPrice / выдёргивали stake вокруг реальных риск-событий
   (FTX ноя-2022, UST, Stream Finance ноя-2025). Если медиана репрайсинга — дни,
   edge жив; если часы — мёртв.
2. **Паттерны покупок кавера перед известными крахами** (адреса-покупатели FTX-каверов
   осени 2022): закрывает вопрос о конкуренции — делал ли это уже кто-то систематически.
3. **Снимок цен/ёмкостей по листингам во времени** (у нас уже есть API-точка от
   2026-07-05) — трекать targetPrice-изменения форвардно, дёшево.
4. Проверка OpenCover-канала и вторичного рынка (read-only, по докам/контрактам).

Всё Gate-1 — read-only он-чейн/API разведка, капитал 0, соответствует дисциплине.
